Sydney Sister


Sydney Sister is an e-commerce fashion store allowing users to browse, add their favourite items to their cart and place an order. Targeted at females aged 14-30, the main objective of this project was to create a Ruby on Rails web application focusing on the demand for retail stores to extend their customer traffic to a broader market.
Features include welcome email upon sign up, shopping cart, and placing an order using Stripe.

Animal Rescue


Animal Rescue allows cats and dogs to be surrendered and rehomed. You can keep track of the animals you have welcomed into your world and/or surrended with your dashboard.
Built as a Ruby on Rails web application, features include RSpec and RSpec Cabybara, Bcrypt, Haml, Bootstrap Carrierwave, and AWS.

Travel Trax


Travel Trax is a Ruby on Rails web application built to help you keep track of your travels, whether it be a daily adventure or a trip across the world. Add your country, add as many blog posts as you like and don't forget to locate them.
Features include Geocoder, Geocomplete, Bootstrap and Bcrypt.
